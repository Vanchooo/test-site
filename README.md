# JDI React/Next.js/Storybook example

## How to use

Clone the repo
Install dependencies:

```sh
npm install
```

### Storybook

You can run storybook in development mode

```sh
npm run storybook
```

Or build it staticly

```sh
npm run build-storybook
npx http-server storybook-static
```

### Next.js

```sh
npm run dev
```
